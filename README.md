# Burp-It

🍔 BURP-IT: Food Delivery & Order Management System in C BURP-IT is a console-based food ordering and management system built in C. It supports both customer and admin interfaces, providing features like registration, login, menu management, order placement and cancellation, payment processing, and viewing order history.


📋 Features

👤 User Authentication Register as Customer or Admin (admin registration requires a secure key).

Secure login with password masking using getch().

🍽️ Menu Management (Admin) Add, update, remove food items.

View complete menu with stock and pricing.

🛒 Customer Panel View menu and place orders.

Process payments (Cash or Card).

View "About Us" section.

🧾 Admin Panel Manage menu items.

Process orders with priority queue (urgent orders first).

View all current orders.

View complete order history and payment logs.

💳 Payment System Cash and Card.

Payment logging with amount and method.

📚 Order History Maintains a complete record of all processed orders.

Total revenue display.


🛠️ Technologies Used C Language

File Handling (.txt files for data persistence)

Data Structures: Linked List (Menu), Queue (Order Queue), Priority Queue (Urgent Orders)
